# Chrome New Tab Redirect

A simple Chrome extension that redirects new tabs to a URL of your choice.

## Features

- Set any URL as your new tab page
- Quick presets: Google, DuckDuckGo, GitHub, YouTube, or blank page
- Clean, minimal settings interface
- No tracking, no analytics, no permissions except storage

### Manual Installation
1. Download or clone this repository
2. Open `chrome://extensions/` in Chrome
3. Enable "Developer mode" (top right)
4. Click "Load unpacked"
5. Select the extension folder

## Usage

1. Click the extension icon or go to `chrome://extensions/` → Extension Options
2. Enter your desired URL
3. Click Save
4. Open a new tab — it will redirect to your URL
