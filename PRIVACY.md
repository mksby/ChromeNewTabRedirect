# Privacy Policy

## Chrome New Tab Redirect

**Last updated:** January 2026

### Data Collection

This extension does **not** collect, store, or transmit any personal data.

### Local Storage

The extension stores only one piece of information locally on your device:
- The redirect URL you choose to set

This data is stored using Chrome's `storage.sync` API, which means it syncs across your Chrome browsers if you're signed in. This data never leaves your browser and is not accessible to the developer or any third parties.

### Permissions

- **Storage**: Used solely to save your redirect URL preference

### Contact

If you have questions about this privacy policy, please open an issue on GitHub.
